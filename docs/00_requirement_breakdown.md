# LabelHub 需求拆解

## 功能优先级定义

| 优先级 | 定义 | 交付要求 |
|--------|------|----------|
| **P0** | 核心功能，必须完成 | 演示必备，主流程可跑通 |
| **P1** | 进阶功能，重要但非必需 | 提升用户体验和效率 |
| **P2** | 高级功能，锦上添花 | 完善系统能力 |

---

## P0 - 核心功能

### 1. Owner 端功能

#### 1.1 任务管理
- 创建标注任务（任务名称、描述、截止日期）
- 查看任务列表
- 编辑/删除任务
- 发布/暂停/结束任务

#### 1.2 标注模板配置
- 创建标注模板
- 支持多种组件类型：ShowItem、TextInput、Textarea、Radio、Checkbox、TagSelect、RichText、FileUpload、JsonEditor、LLMAssist、Group、Tabs
- 模板预览
- 保存/发布模板

#### 1.3 数据集导入
- 支持 CSV、JSON、JSONL 格式导入
- 导入官方测试数据（qa_quality、preference_compare）
- 数据预览和验证
- 批量导入

#### 1.4 AI 审核配置
- 设置 AI 审核开关
- 配置评分维度和阈值
- 设置 Prompt 模板（Mock 模式）

#### 1.5 结果查看与导出
- 查看标注结果统计
- 导出数据（JSON、JSONL、CSV、XLSX）
- 导出历史记录

#### 1.6 审计日志
- 查看所有关键操作日志
- 支持按用户、操作类型、时间筛选

---

### 2. Labeler 端功能

#### 2.1 任务广场
- 浏览可领取的任务
- 领取任务

#### 2.2 在线标注
- 加载标注模板
- 填写标注内容
- 自动保存草稿（每 30 秒）
- 提交标注

#### 2.3 打回修改
- 查看打回原因
- 修改标注内容
- 重新提交

---

### 3. AI Agent 功能

#### 3.1 自动预审
- 标注提交后自动触发
- Mock 模式生成结构化评分
- 输出结论（pass/reject/human_review）
- 生成质检建议

#### 3.2 结果可追溯
- 记录 prompt_template
- 记录 raw_response
- 记录 parsed_result
- 记录 dimension_scores

---

### 4. Reviewer 端功能

#### 4.1 审核管理
- 查看待审核列表
- 查看 AI 预审结果
- 人工复审/终审

#### 4.2 审核操作
- 通过
- 打回（附带原因）
- 修订（直接修改）

---

## P1 - 进阶功能

### 1. Owner 端
- 任务进度统计图表（数据看板）
- 模板复用和共享
- 字段映射导出

### 2. Labeler 端
- 任务搜索和筛选
- 标注历史记录
- Rubric 标注规则助手

### 3. Reviewer 端
- 批量审核操作
- 审核历史记录
- 标注质量统计

### 4. 系统功能
- BM25 + Vector 规则检索
- 用户登录和角色管理
- 数据权限隔离

---

## P2 - 高级功能

### 1. AI 集成
- 接入真实 LLM API
- Rerank
- GraphRAG / 规则图谱

### 2. 系统扩展
- 支持多租户
- 高级权限系统
- 移动端适配
