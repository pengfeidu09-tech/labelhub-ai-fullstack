# 基础技术文档

## 技术栈

### 前端
- React 18 + TypeScript + Vite
- Ant Design 组件库
- React Router v6 路由管理
- Zustand 状态管理
- @dnd-kit 拖拽库（模板设计器）

### 后端
- FastAPI + Pydantic
- SQLAlchemy ORM
- SQLite 数据库
- Uvicorn ASGI 服务器

### AI
- DashScope OpenAI-compatible API
- 默认模型：qwen3.7-plus
- Demo fallback：Mock Provider（无 API Key 时自动降级）

## 核心设计

### Schema 驱动动态表单
模板 Schema 定义字段类型、校验规则、联动逻辑，FormRenderer 根据 Schema 动态渲染。

### 三层数据结构
Task -> DatasetItem/WorkUnit -> Submission/AIReviewRun/HumanReview/AuditLog

### 状态机流转
数据项经历：未领取 -> 已领取 -> 标注中 -> 已提交 -> AI预审中 -> AI已预审 -> 人工审核中 -> 已通过/已打回 -> 可导出

### 高性能标注工作台
- formData ref + 500ms debounce 防止每次击键触发重渲染
- 隔离计时器组件（1s interval 替代 60fps RAF）
- React.memo 隔离 Rubric 面板和 LLM 结果面板
- 移除 StrictMode 避免开发模式双重执行

### 全链路审计
所有关键操作（创建任务、模板保存、提交、AI 预审、审核、导出）写入审计日志。
