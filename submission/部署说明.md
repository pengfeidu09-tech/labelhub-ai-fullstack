# 部署说明

## 环境要求
- Node.js >= 18
- Python >= 3.10
- pip

## 后端启动

```bash
cd backend
pip install -r requirements.txt

# 配置环境变量
cp ../.env.example .env
# 编辑 .env 填入 DASHSCOPE_API_KEY 等

# 启动服务
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

后端默认端口：8000
数据库：SQLite（自动创建 backend/labelhub.db）

## 前端启动

```bash
cd frontend
npm install

# 启动开发服务器
npm run dev
```

前端默认端口：5173
API 代理：已配置 Vite proxy 指向 http://127.0.0.1:8000

## AI Provider 配置

默认使用 DashScope qwen3.7-plus 模型。

在 `.env` 中配置：
```
AI_PROVIDER=dashscope
DASHSCOPE_API_KEY=your_key
DASHSCOPE_MODEL=qwen3.7-plus
```

如果 API Key 未配置，系统会自动 fallback 到 Mock 模式，保证 Demo 可用。

## 验证启动

1. 访问 http://localhost:5173 看到 LabelHub 首页
2. 访问 http://localhost:8000/api/health 确认后端健康
3. 选择角色进入系统
