# 演示视频说明

## 视频概述
LabelHub 10 分钟全流程演示，覆盖官方参考流程的 5 段角色链路。

## 演示主线
任务 #10 官方原题·问答质量标注

## 扩展演示
任务 #11 官方原题·偏好对比标注

## 覆盖角色
- 任务负责人：创建任务、搭建模板、配置审核规则、发布任务
- 标注员：任务市场领取、标注工作台作答、LLM 辅助、草稿保存、提交
- AI 审核 Agent：提交后自动预审、多维度评分、综合判定
- 人工审核员：审核队列、审核详情、通过或打回
- Owner / 数据落地：结果中心、多格式导出、审计追踪

## 时间轴
| 时间段 | 内容 |
|--------|------|
| 0:00-1:00 | 项目概述与仪表盘总览 |
| 1:00-2:30 | 任务负责人：创建任务、搭建模板、配置审核规则 |
| 2:30-5:00 | 标注员：任务市场、标注工作台、Rubric 参考、LLM 辅助 |
| 5:00-6:00 | 提交答案与 AI Agent 自动预审 |
| 6:00-7:30 | 人工审核员：审核队列、审核详情、通过/打回 |
| 7:30-9:00 | Owner：结果中心、多格式导出、审计日志 |
| 9:00-10:00 | 性能验证与总结 |

## 关键演示点
- 只读 Rubric 标准参考（非交互式）
- LLM 辅助仅点击触发，不自动运行
- 审核规则配置独立于 Rubric 标准参考
- AI 预审不替代人工审核，辅助分流
- 高性能标注工作台（CPU 正常，无卡顿）
