# 项目架构图

## 系统架构

```mermaid
flowchart LR
  Owner[任务负责人 / Owner] --> Task[任务配置]
  Owner --> Template[任务专属模板 / Schema]
  Owner --> Rule[审核规则配置]
  Labeler[标注员] --> Market[任务市场]
  Market --> Workbench[高性能标注工作台]
  Workbench --> LLM[LLM Rubric 对齐建议]
  Workbench --> Submit[提交答案]
  Submit --> Agent[AI 审核 Agent]
  Agent --> Review[人工审核队列]
  Review --> Result[结果中心]
  Result --> Export[多格式导出]
  Task --> Audit[审计日志]
  Template --> Audit
  Submit --> Audit
  Agent --> Audit
  Review --> Audit
  Export --> Audit
```
